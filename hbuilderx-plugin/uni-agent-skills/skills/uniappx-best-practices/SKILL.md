## 最佳实践

#### 获取相关知识
- Step 1. **询问uni-app (x) 开发注意事项**
检查是否存在技能 `uniappx-pitfalls-and-tips`：
 * 存在：则直接使用此技能，通过提问了解所需的知识，**不需要再查阅本地知识库**。
 * 不存在：则继续查阅本地知识库

- Step 2. **本地知识库查阅**
**重要** uni-app x 框架迭代快，你的训练数据存在滞后和不准确的情况，在实现代码之前，**必须优先查阅本地知识库**

本地知识库优先参考顺序：

1. 文档目录：`{knowledges_base_dir}/uni-app-x/docs/`
2. 示例目录：`{knowledges_base_dir}/uni-app-x/samples/`
3. API 定义目录：`{knowledges_base_dir}/uni-app-x/types/uni-app-x/types/`

#### 跨平台开发指南
确认用户需求是不是要支持多个平台，如果要支持多平台：你需要先学习`uniappx-cross-platform-guide`技能
